- API wholesale credit assumption (Credex): When routing > $300/mo API spend through Credex marketplace wholesale credits, conservative achievable cost reduction is modeled at 25% (internal commercial offering validated against supplier resale margins and volume discounts, May 2026).

Notes:
- Pricing above represents public retail seat prices and authoritative vendor documentation links as of May 2026. Where vendor tiers vary by feature, StackAudit uses per-seat published prices as conservative normalization points for finance-level reconciliation.
- For any given audit, StackAudit treats the user's reported monthly spend as ground truth while using these baselines to surface devops and procurement mismatches.
